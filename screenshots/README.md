# Screenshots Directory

This directory contains screenshots for LinkedIn portfolio showcase.

## Recommended Screenshots

1. **Dashboard Overview** - Main production interface
2. **Template Management** - CRUD operations
3. **History & Export** - Session history with Excel export
4. **Mobile View** - Responsive design on mobile

## Instructions

1. Open the app in browser: `npx serve .`
2. Navigate to each feature
3. Take screenshots (1920x1080 recommended)
4. Save as: `feature-name.png`
