## 2. Feature Description

Implemented optional efficiency tracking in production templates. Users can now choose whether to enable efficiency calculation when creating a template.

### 2.1 Behavior

- **Efficiency Enabled (default)**: Shows hours/minutes dropdowns, theoretical time, and efficiency percentage
- **Efficiency Disabled**: Only shows total pieces and meters produced
